# 任务目标：抓取XX论坛问答区帖子与评论（Python）

## 1. 核心任务
我需要一个脚本，能够抓取指定论坛板块的所有问答帖子内容以及每个帖子下的所有评论。

## 2. 目标网址
- 板块首页URL：https://www.dongchedi.com/community/4363/wenda

## 3. 具体抓取要求
1.  **遍历所有页面**：脚本需要能自动识别并点击“下一页”，直到抓取完该板块的所有页面。
2.  **抓取帖子列表**：在每个列表页，需要抓取所有帖子的标题和帖子的链接。
3.  **进入帖子详情页**：对于每个帖子，脚本需要访问其链接，进入详情页。
4.  **抓取主贴内容**：
    *   帖子标题
    *   作者
    *   发布时间
    *   帖子正文内容
5.  **抓取所有评论**：
    *   评论者
    *   评论时间
    *   评论内容
    *   （如果评论也有分页，也需要处理）

## 4. 输出格式
请将所有抓取到的数据，最终整理并保存为一个 **JSON 文件 (`output.json`)**。
格式要求如下：

[
  {
    "title": "帖子标题1",
    "author": "作者A",
    "publish_time": "2024-10-26",
    "content": "这是帖子的正文内容...",
    "comments": [
      {
        "commenter": "评论者X",
        "comment_time": "2024-10-27",
        "comment_content": "这是一条评论。"
      },
      {
        "commenter": "评论者Y",
        "comment_time": "2024-10-28",
        "comment_content": "这是另一条评论。"
      }
    ]
  },
  {
    "title": "帖子标题2",
    ...
  }
]


## 5. 技术栈建议
- 建议使用 **Python** 语言。
- 对于网页抓取，请根据网站类型选择合适的库：
  - 如果是静态网站，可以使用 `requests` + `BeautifulSoup4`。
  - 如果网站内容是动态加载的（需要滚动或点击才出现），请使用 `Selenium` 或 `Playwright` 来模拟浏览器操作。
- 请在代码中添加适当的注释和延时，避免因请求过快被网站封禁。
