# 懂车帝Model Y车友圈问答区爬虫

一个专门用于抓取懂车帝Model Y车友圈问答区帖子和评论的Python爬虫工具。

## ✨ 功能特点

- 🚀 **全自动抓取**: 自动遍历所有页面，无需手动干预
- 📝 **完整数据**: 抓取帖子标题、作者、发布时间、正文内容和所有评论
- 🔄 **智能翻页**: 自动识别并点击"下一页"，支持多种翻页方式
- 💾 **数据备份**: 定期自动备份，防止数据丢失
- 🛡️ **反检测**: 内置多种反爬虫检测机制
- ⚙️ **高度可配置**: 支持自定义延时、重试次数等参数
- 📊 **统计报告**: 自动生成抓取统计报告
- 🔧 **错误处理**: 完善的异常处理和重试机制

## 🎯 目标网站

- **目标URL**: https://www.dongchedi.com/community/4363/wenda
- **论坛**: 懂车帝Model Y车友圈问答区
- **内容**: 问答帖子及其评论

## 📋 抓取内容

### 帖子信息
- 帖子标题
- 作者姓名
- 发布时间
- 帖子正文内容

### 评论信息
- 评论者姓名
- 评论时间
- 评论内容

## 🚀 快速开始

### 环境要求

- Python 3.7+
- Chrome浏览器
- 网络连接

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行爬虫

```bash
# 完整抓取
python scraper.py

# 测试运行（只抓取少量数据）
python test_scraper.py
```

## ⚙️ 配置选项

编辑 `config.py` 文件来自定义爬虫行为：

```python
# 延时配置
DELAY_CONFIG = {
    'min_delay': 2,      # 最小延时（秒）
    'max_delay': 5,      # 最大延时（秒）
}

# 限制配置
LIMITS = {
    'max_pages': 100,    # 最大抓取页数
}

# 浏览器配置
BROWSER_CONFIG = {
    'headless': False,   # 是否无头模式
    'window_size': (1920, 1080),
}
```
